# Лабораторна робота №9: Робота з screen та tmux

## Робота з `screen`

### Запустити команду screen
Для запуску нової сесії `screen` просто введіть команду:
```bash
screen
```
Це створить нову сесію та відкриє в ній оболонку.

### Вийти з сесії screen із завершенням роботи програми
Щоб завершити сесію `screen` (і всі процеси в ній), закрийте всі оболонки в сесії. Найпростіший спосіб — ввести команду `exit` або натиснути `Ctrl+d`.
```bash
exit
```

### Запустити screen з іменем 'current-screen'
Використовуйте прапорець `-S` для присвоєння імені сесії при її створенні.
```bash
screen -S current-screen
```

### Перевірити ім'я сесії та перейменувати її
1.  **Перевірити ім'я:** Ззовні сесії виконайте `screen -ls`.
2.  **Перейменувати:** Ви можете перейменувати сесію ззовні або зсередини.
    *   **Ззовні:**
        ```bash
        screen -S current-screen -X sessionname current-screen-2
        ```
    *   **Зсередини:** Натисніть `Ctrl+a`, потім `:`, і введіть:
        ```
        sessionname current-screen-2
        ```

### Відкрити 4 оболонки та запустити команди
1.  У першому вікні запустіть `top`.
2.  Створіть нове вікно: `Ctrl+a` `c`.
3.  У новому вікні запустіть `tail -f /var/log/messages`.
4.  Створіть ще одне вікно: `Ctrl+a` `c`.
5.  Запустіть `dnf update`.
6.  Створіть останнє вікно: `Ctrl+a` `c`.
7.  Запустіть `ls`.

Для навігації між вікнами використовуйте `Ctrl+a` `n` (наступне), `Ctrl+a` `p` (попереднє) або `Ctrl+a` `"` (список вікон).

### Вийти із сесії, не завершуючи її (detach)
Щоб від'єднатися від сесії, залишивши всі процеси працювати, натисніть `Ctrl+a` `d`.

### Переглянути список сесій та підключитися
1.  **Переглянути список:**
    ```bash
    screen -ls
    ```
2.  **Підключитися до сесії:**
    ```bash
    screen -r current-screen-2
    ```

### Розділити робочу область
1.  Перебуваючи у вікні з `top`, натисніть `Ctrl+a` `S` (велика 'S') для горизонтального розділення.
2.  Перейдіть на нову панель: `Ctrl+a` `Tab`.
3.  Переключіться на вікно з `tail -f /var/log/messages`: `Ctrl+a` `n` (або `p`).

## Робота з `tmux` (аналогічні дії)

### Запустити команду tmux
```bash
tmux
```
Або для створення іменованої сесії:
```bash
tmux new-session -s my_session
```

### Вийти з сесії tmux із завершенням роботи
Як і в `screen`, введіть `exit` або натисніть `Ctrl+d`.
```bash
exit
```

### Запустити tmux з іменем 'current-session'
```bash
tmux new-session -s current-session
```

### Перевірити ім'я сесії та перейменувати її
1.  **Перевірити ім'я:**
    ```bash
    tmux list-sessions
    ```
2.  **Перейменувати:**
    ```bash
    tmux rename-session -t current-session current-session-2
    ```
    Або зсередини сесії: `Ctrl+b` `$` та введіть нове ім'я.

### Відкрити 4 оболонки та запустити команди
1.  У першому вікні запустіть `top`.
2.  Створіть нове вікно: `Ctrl+b` `c`.
3.  У новому вікні запустіть `tail -f /var/log/messages`.
4.  Створіть ще одне вікно: `Ctrl+b` `c`.
5.  Запустіть `dnf update`.
6.  Створіть останнє вікно: `Ctrl+b` `c`.
7.  Запустіть `ls`.

Для навігації між вікнами використовуйте `Ctrl+b` `n` (наступне), `Ctrl+b` `p` (попереднє) або `Ctrl+b` `w` (список вікон).

### Вийти із сесії, не завершуючи її (detach)
Натисніть `Ctrl+b` `d`.

### Переглянути список сесій та підключитися
1.  **Переглянути список:**
    ```bash
    tmux ls
    ```
2.  **Підключитися до сесії:**
    ```bash
    tmux attach-session -t current-session-2
    ```
    Або скорочено: `tmux a -t current-session-2`

### Розділити робочу область
1.  Перебуваючи у вікні з `top`, натисніть `Ctrl+b` `"` для горизонтального розділення або `Ctrl+b` `%` для вертикального.
2.  Перейдіть на нову панель за допомогою стрілок: `Ctrl+b` `↑`/`↓`/`←`/`→`.
3.  У новій панелі переключіться на потрібне вікно (`Ctrl+b` `w` і виберіть) або просто запустіть команду `tail -f /var/log/messages`.
