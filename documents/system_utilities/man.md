Команда `man` (manual) — це вбудована довідкова система в Linux. Вона надає доступ до "сторінок керівництва" (manual pages) для практично кожної команди, утиліти, системного виклику або конфігураційного файлу.

Якщо ви не знаєте, як працює команда або що означає її опція, `man` — це перше місце, куди варто звернутися.

#### **Базовий синтаксис**

```bash
man [розділ] назва_сторінки
```

*   **`назва_сторінки`**: Зазвичай це назва команди (наприклад, `ls`, `grep`).
*   **`[розділ]`**: (Опціонально) Номер розділу, в якому шукати сторінку.

--- 

### **Розділи `man`-сторінок**

Керівництво розділене на секції за тематикою:

| Розділ | Опис |
| :--- | :--- |
| **1** | Команди користувача та виконувані програми. |
| **2** | Системні виклики (функції ядра). |
| **3** | Бібліотечні виклики (функції з бібліотек, наприклад, `printf` з `libc`). |
| **4. ** | Спеціальні файли (зазвичай у `/dev`). |
| **5** | Формати файлів та конвенції (наприклад, `man 5 passwd` покаже структуру файлу `/etc/passwd`). |
| **6** | Ігри. |
| **7** | Різне (огляди, конвенції, наприклад, `man 7 regex`). |
| **8** | Команди системного адміністрування (зазвичай потребують `sudo`). |

За замовчуванням `man` шукає по всіх розділах, починаючи з першого.

--- 

### **Навігація всередині `man`**

Інтерфейс `man` використовує навігацію, схожу на `less`.

| Клавіша | Дія |
| :--- | :--- |
| **Стрілки вгору/вниз** | Прокрутка по одному рядку. |
| **`Пробіл`** (Spacebar) | Прокрутка на один екран уперед. |
| **`b`** | Прокрутка на один екран назад. |
| **`/шаблон`** | Пошук тексту `шаблон` (вперед). |
| **`?шаблон`** | Пошук тексту `шаблон` (назад). |
| **`n`** | Перейти до наступного знайденого результату. |
| **`N`** | Перейти до попереднього знайденого результату. |
| **`q`** | Вийти з `man`. |

--- 

### **Приклади використання**

1.  **Отримати довідку по команді `ls`:**
    ```bash
    man ls
    ```

2.  **Дізнатися про конфігураційний файл `fstab`:**
    Файли конфігурації знаходяться в 5-му розділі.
    ```bash
    man 5 fstab
    ```

3.  **Коли імена збігаються:**
    Існує команда `printf` (розділ 1) і бібліотечна функція `printf` (розділ 3). Щоб вказати, яка саме сторінка потрібна, вказують номер розділу.
    ```bash
    man 1 printf
    man 3 printf
    ```

4.  **Пошук сторінок за ключовим словом:**
    Якщо ви не знаєте точної назви команди, але знаєте, що вона робить.
    ```bash
    man -k "copy files"
    ```
    *Або еквівалентна команда `apropos "copy files"`. Вона покаже список сторінок, в описі яких є ця фраза.*

5.  **Відкрити сторінку в браузері:**
    Деякі системи дозволяють відкрити `man`-сторінку у вигляді HTML.
    ```bash
    man -Hfirefox ls
    ```

`man` — це фундаментальний навик для ефективної роботи в командному рядку Linux.
